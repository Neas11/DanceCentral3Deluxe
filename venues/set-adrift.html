Venue Description

Set Adrift is a prominent dance venue featured in Dance Central 2.

Visual and Functional Design

  • Theme: The venue is a luxury yacht set in a tropical environment.
  • Dynamic Backgrounds: During gameplay, the venue alternates between two distinct locations: the main deck of the yacht and the yacht's helipad.
  • Associated Crew: It is the home base for Lu$h Crew, which consists of returning characters Miss Aubrey and Angel. The crew is portrayed as wealthy and fashionable, reflecting the luxurious nature of the yacht.
